# Python Basics

## 1. What is Python?

Python is a **high-level, interpreted, and general-purpose programming language** created by **Guido van Rossum** and released in **1991**.

It is one of the most popular programming languages because of its **simple syntax**, **easy readability**, and **wide range of applications**. Python is beginner-friendly and also powerful enough for professional software development.

Python follows an easy-to-read syntax similar to the English language, which makes coding easier to understand and write.

Python supports multiple programming paradigms such as:

* Procedural Programming
* Object-Oriented Programming (OOP)
* Functional Programming

Python is widely used in:

* Web Development
* Artificial Intelligence
* Machine Learning
* Data Science
* Automation
* Cybersecurity
* Cloud Computing
* Software Development

### Features of Python

* Easy to learn and understand
* Open-source and free to use
* Cross-platform compatibility
* Large community support
* Huge collection of libraries and frameworks
* Easy database integration
* Fast development process

### Example Python Program

```python
print("Hello World")
```

**Output:**

```text
Hello World
```
