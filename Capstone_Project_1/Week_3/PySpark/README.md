# Employee Attendance and Productivity Tracker - Week 3

## Overview

This project uses PySpark to analyze large employee attendance logs and productivity data.

## Technologies Used

* Python
* PySpark

## Features

* Load large attendance dataset from CSV
* Calculate employee work hours
* Calculate productivity scores
* Filter late logins
* Identify absentees
* Group data by department
* Calculate average work hours by department
* Calculate average productivity by department

## Files

* large_attendance.csv
* attendance_pyspark.py (or attendance_analysis_pyspark.ipynb)

## Output

* Late login report
* Absentee report
* Department-wise work hour summary
* Department-wise productivity summary
