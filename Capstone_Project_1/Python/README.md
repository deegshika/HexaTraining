# Employee Attendance and Productivity Tracker - Week 2

## Overview

This module analyzes employee attendance and productivity data using Python, Pandas, and NumPy.

## Technologies Used

* Python
* Pandas
* NumPy

## Features

* Read attendance data from CSV
* Clean missing or invalid records
* Calculate employee work hours
* Calculate break times
* Calculate productivity scores
* Identify top performers
* Identify bottom performers
* Generate department-wise summary
* Export cleaned dataset

## Files

### attendance.csv

Input dataset containing employee attendance and task information.

### attendance_analysis.py

Python script used for data cleaning and analysis.

### cleaned_attendance.csv

Processed dataset generated after analysis.

## Output

The script generates:

* Work hours for each employee
* Productivity scores
* Top performers
* Bottom performers
* Department performance summary
