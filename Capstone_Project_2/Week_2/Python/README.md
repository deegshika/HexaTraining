# Retail Sales Analytics System - Week 2

## Overview

This module performs sales data analysis using Python, Pandas, and NumPy. It processes retail sales records, calculates business metrics, and generates performance summaries for products and stores.

## Technologies Used

- Python
- Pandas
- NumPy

## Features

- Load sales data from CSV
- Clean missing values
- Correct data types
- Calculate revenue
- Calculate profit
- Calculate discount percentages
- Calculate profit margins
- Generate product-wise revenue summary
- Generate store-wise revenue summary
- Identify top-performing products
- Identify underperforming products

## Files

- sales.csv
- sales_analysis.py
- cleaned_sales.csv

## Output

- Cleaned sales dataset
- Revenue calculations
- Profit calculations
- Product performance summary
- Store performance summary
- Top and underperforming product report
