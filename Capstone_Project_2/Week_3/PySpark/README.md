# Week 3 - Retail Sales Analysis using PySpark

## Project Overview
This project analyzes large retail sales data using PySpark.  
It helps identify underperforming products and calculate average monthly revenue for each store.

## Files Included
- `retail_sales_pyspark.ipynb` → PySpark notebook for analysis
- `large_sales.csv` → Large sales dataset used as input

## Tasks Performed
- Loaded large sales data into PySpark
- Calculated revenue and profit
- Filtered underperforming products based on low revenue or high returns
- Grouped sales data by store and month
- Calculated average monthly revenue for each store
- Saved the output files as CSV

## Output
The project generates:
1. Underperforming products report
2. Store-wise average monthly revenue report
