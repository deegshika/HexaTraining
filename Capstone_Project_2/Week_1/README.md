# Retail Sales Analytics System

## Overview

This project is designed to collect and analyze retail sales data across multiple stores and products. The system helps monitor store performance and identify underperforming products.

## Technologies Used

* MySQL
* MongoDB

## Features

### MySQL

* Stores Management
* Products Management
* Employees Management
* Sales Tracking
* CRUD Operations
* Stored Procedure for Daily Sales Calculation

### MongoDB

* Promotional Campaign Feedback Storage
* Product-based Search
* Region-based Search
* Indexing for Faster Queries
